# DPanel

DPanel is a web interface leveraging FastAPI, Redis, Go PubSub, and Nginx to manage Docker processes, including containers, images, and volumes, with real-time statistics.

### Technology Stack:
- Python
- JavaScript
- Go
- Nginx
- FastAPI
- Reddis
